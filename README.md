# 🍕 Pizza Sales Analysis

This project uses SQL queries to analyze pizza sales data and uncover business insights.

## 📊 Key Insights
- Total Orders: 21,350
- Total Revenue: $817,860.05
- Highest-Priced Pizza: The Greek Pizza ($35.95)
- Most Common Size: Large (18,526 orders)
- Top 5 Pizzas: Classic Deluxe, Barbecue Chicken, Hawaiian, Pepperoni, Thai Chicken
- Peak Hours: 12 PM – 1 PM, 5 PM – 7 PM
- Cumulative Revenue Trend: Steady growth over time

## 🛠️ Skills Demonstrated
- SQL Joins, Aggregations, Window Functions
- Business Intelligence & Data Analytics
- Translating raw data into actionable insights
